# UI for Site Scraping - Made with Vue CLI

Create `.env.local` and add:
```
VUE_APP_API_BASE_URL=http://localhost:3037
```

---

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
